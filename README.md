# Untitled Incremental Lootbox Game

A dopamine rich incremental lootbox game built with Rust and Ratatui.

## Demo

![Demo](./demo.gif)

## Features

- Incremental lootbox opening mechanics
- Relics, Skills, Upgrades
- Terminal-based UI
- Saved Progress
- Dopamine

## Installation

```bash
cargo build --release
```

## Usage

**Native (recommended):**
```bash
cargo run --release
```
